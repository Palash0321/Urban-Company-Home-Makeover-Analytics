# Urban-Company-Home-Makeover-Analytics
End-to-end SQL Server and Power BI analytics project analyzing 3,168 home makeover leads and ₹15.64 Cr revenue across Bengaluru. Features KPI reporting, sales funnel, regional analysis, project manager performance, customer insights, and interactive dashboards.

# Urban Company Home Makeover Analytics Dashboard

## Project Overview

This project is an end-to-end Business Intelligence solution developed using **SQL Server** and **Power BI** to analyze operational and sales performance in the **Home Makeover (Painting & Renovation)** category.

The project simulates a real-world analytics environment using a dataset of **3,168 customer leads** across Bengaluru. It demonstrates SQL-based data analysis, KPI reporting, dashboard development, customer analytics, regional performance analysis, and operational insights for business decision-making.

---

## Business Problem

Home makeover operations involve multiple stakeholders, regions, project managers, and service categories. Business teams require centralized reporting to monitor:

* Lead generation
* Revenue performance
* Lead-to-Hire conversion
* Regional demand
* Project Manager productivity
* Key Account Manager performance
* Service demand
* Customer insights

This dashboard provides interactive reporting to help stakeholders identify growth opportunities and monitor operational performance.
